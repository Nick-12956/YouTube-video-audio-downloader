# YouTube-video-audio-downloader
This application has self-contained UI.  
To run the application in computer, download ZIP file from "Releases" tab, unpack it, and simply run the "source file" in it.

# Application Roles
1. It has option to download single file/multiple files at ones.
2. It has option to download video(mp4) / audio(mp3) file.
3. It has option to choose resolution for video files.
4. It has option to choose download folder (default : downloads folder)
5. It has option to choose window theme (system/light/dark)
6. It shows progress bar with percentage download and title of the file while downloading.
7. It shows the download logs(status) of each file after downloading for multi-file download option.
